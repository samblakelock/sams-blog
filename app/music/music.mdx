## Learn guitar

| Course                                                                                                        | What you'll learn                                                                                               |
| ------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------- |
| [Soloing Learning Pathway](https://www.pickupmusic.com/master-classes/soloing-learning-pathway)               | Intermediate / advanced class on improvisation, scales and theory.                                              |
| [Jazz Bootcamp](https://www.pickupmusic.com/master-classes/jazz-bootcamp)                                     | Intro to jazz guitar taught by me and Cecil Alexander. It's an older class but has a lot in it.                 |
